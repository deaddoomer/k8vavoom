// Default game detection configuration file
// first param is also a game name, if "gamename" is absent
// "gamename" is what will be put into "game_name" cvar


// Doom II: Hell On Earth
game basev/doom2
base basev/doom
iwad doom2.wad
altiwad doom2f.wad
param -doom2
gamename doom2
warp MAP##
filter doom
filter game-doom
filter game-doomstrifechex
filter game-doomchex
filter doom.doom2
filter doom.doom2.commercial
end
